# Usage

```bash
npx @blu1606/create-walrus-app [project-name] [options]
Options:

--sdk <mysten|tusky|hibernuts> - SDK to use
--framework <react|vue|plain-ts> - Frontend framework
--use-case <simple-upload|gallery|defi-nft> - Application template
-p, --package-manager <npm|pnpm|yarn|bun> - Package manager (optional)
--skip-install - Skip dependency installation
--skip-git - Skip git initialization
--skip-validation - Skip project validation
